# RESP.app 文档

**RESP.app** (原 RedisDesktopManager) 是一个跨平台的开源 Redis® 图形管理工具，可在 Windows、Linux 和 macOS 上运行。该工具为您提供了一个易于使用的 GUI 来访问您的 Redis® 数据库并执行一些基本操作：以树形视图查看键、CRUD 键、通过 Shell 执行命令。RESP.app 支持 SSL/TLS 加密、SSH 隧道和云端 Redis 实例，例如：Amazon ElastiCache、Microsoft Azure Redis 缓存和其他 Redis® 云服务。
